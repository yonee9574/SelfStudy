# Chapter 11 : 트랜지션과 애니메이션

## transform 과 변형함수

CSS 에서 변형을 적용할려면 transform 속성과 변형 함수이름을 함께 작성해야 한다.

- [ 기본형 ]  transform : 함수

```html
.photo { transform: translate(50px, 100px); }
```

## 트랜지션 ( transition )

웹 요소의 배경색을 바꾸거나 도형 테두리를 사각형에서 원형으로 바꾸는 것처럼 스타일 속성이 바뀌는 것을 말함.
