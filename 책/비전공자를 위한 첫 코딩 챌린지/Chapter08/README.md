요소 : 텍스트 사진등 보여줄 대상

패딩 : 요소 주변 영역을 감싸고있음 ( 해당부분은 투명 )

테두리 : 요소와 패딩을 감싸는 테두리

#### 쉽게 이해하는 방법

마진은 물건을 팔았을 때 남는 이익 / 따라서 요소,패딩,테두리를 표시하고 남은 영역이 마진
