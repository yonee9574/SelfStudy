HTML이란?

HTML (Hypertext Markup Language,하이퍼텍스트 마크업 언어)는 프로그래밍 언어는 아니고,
우리가 보는 웹페이지가 어떻게 구조화되어 있는지 브라우저로 하여금 알 수 있도록 하는 마크업 언어

HTML 태그들을 알아보자

● < small > 태그

작은 텍스트를 만들때 사용한다
예)
 안녕하세요 저는 소심한 이연입니다.

"소심한" 이라는 글자를 표현을 할때 < small > 태그를 사용해 보았다.

● < sub >< sup > 태그

윗첨자 아래첨자 태그
주로 수학식, 화학기호를 표현하거나 단락에서 주석을 표시할 때 많이 쓰이는 방식

예1)
 탄산나트륨(NA2CO3)은 유리에 자외선 차단도 하고 여러가지 좋은 기능을 한다.

예2)
 8=4x2+16x+24 ---

● < ins > 태그

텍스트 아래쪽에 라인(line)을 추가하여 문서에 추가된 텍스트를 표현할 때 사용

예)
순천대학교 정보통신학과 이명배 교수님은
수업시간에 열정적으로 강의하시는걸로 유명하시다.

● < del > 태그

텍스트 한가운데 라인(line)을 추가하여 문서에서 삭제된 텍스트를 표현할 때 사용

예)
 안녕하세요 저는 소심한이제는 더 이상 소심하지 않은 이연입니다.

● < mark > 태그

형광펜으로 칠한 것처럼 하이라이트된(highlighted) 텍스트를 정의할 때 사용

예)
 순천대학교 정보통신공학과 편입생 4학년 20187090 이연입니다.
